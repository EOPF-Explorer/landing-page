Zarr Conventions (Released) â€‹

The three foundational conventions released at the Zarr Summit Rome, October 2025:

ConventionDescriptionRepository
MultiscalesEncoding multiscale pyramid information for data stored in Zarr groupszarr-conventions/multiscales
Projection (geo-proj)Encoding datum and coordinate reference system (CRS) information for geospatial datazarr-conventions/geo-proj
SpatialDescribing the relationship between array indices and spatial coordinateszarr-conventions/spatial

Data Model Specifications â€‹

V0: Maximalist Compatibility (September 2025)

Following GeoZarr 0.4:

  • Zarr v3
  • Climate and Forecast (CF) conventions integration
  • Grid mapping 0D array implementation
  • Simple CRS + Affine transform approach
  • GDAL compatibility in v3.12

V1: Zarr Conventions (January 2026)

  • Multiscales Zarr convention
  • Projection Zarr convention
  • Spatial Zarr convention
  • Full GDAL support path
